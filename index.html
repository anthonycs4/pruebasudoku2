¿How to play?

Sudoku is a puzzle that consists of placing numbers from 1 to 9 in the blank squares of a large grid, generally 81 squares, subdivided into smaller squares of 9 squares (three on each side), with the aim of that in each row, each column or each 3 x 3 square, no number is repeated. Difficulty is defined by the number of completed grids at the start of each game.
